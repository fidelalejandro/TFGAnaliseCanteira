# Celta Canteira

Ferramenta desenvolvida como parte dun Traballo Fin de Grao para apoiar a análise de xogadores de canteira do RC Celta de Vigo.

O sistema permite consultar información de xogadores, comparar perfís físicos, xerar rankings de similitude e obter explicacións en linguaxe natural a partir de datos previamente procesados en Databricks. A ferramenta combina un backend en Python, unha interface web con Gradio, visualizacións interactivas con Plotly, orquestración mediante LangChain/LangGraph e xeración textual con Claude Haiku.

O núcleo analítico baséase en algoritmos de matching e clasificación que empregan centroides de perfís, relevancia de variables, distancia euclídea ponderada, similitude do coseno e clasificación baseada no acordo entre métodos.

## Tecnoloxías principais

- Python
- Databricks
- Gradio
- Plotly
- LangChain
- LangGraph
- Claude Haiku
- Anthropic API

## Aviso

Este repositorio non inclúe datos reais de xogadores nin credenciais de acceso. Os datos deportivos utilizados polo sistema pertencen ao RC Celta de Vigo e non forman parte do código publicado.
